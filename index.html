退款政策

一、數位商品性質說明

本服務提供之 eSIM 為數位型電信服務，購買後以 QR Code 方式交付。QR Code 一經掃描安裝至裝置，即視為服務已完整提供。

二、猶豫期適用說明

依據《消費者保護法》第 19 條及《通訊交易解除權合理例外情事適用準則》第 2 條第 5 款規定，本商品屬「一經提供即為完成之線上服務」，於消費者啟用（掃描 QR Code 安裝）後，不適用 7 日猶豫期退貨規定。

三、可申請退款的情況

  • 已付款但尚未收到 QR Code（可申請全額退款）
  • 已收到 QR Code 但尚未掃描啟用，於 24 小時內提出申請（可申請全額退款）
  • eSIM 因技術瑕疵無法正常啟用，經確認非裝置相容性問題（提供補發或退款）
  • 服務啟用後發生嚴重網路異常，且非當地電信業者維護造成（依實際情況協商退款）

四、不予退款的情況

  • QR Code 已掃描啟用並正常使用
  • 因消費者裝置不支援 eSIM 或不相容而無法使用
  • 消費者自行操作錯誤導致 QR Code 失效
  • 超出流量上限後的網速降速屬正常規格，不予退款

五、申請方式

請於符合退款條件的情況下，以購買時留存的 Email 聯繫客服，並說明訂單編號及退款原因，我們將於 3 個工作日內回覆處理結果。
